# 이메일 인증 API

Base URL: `/api/email/verification`

회원가입 전 이메일 소유 확인을 위한 인증 코드 발송 및 검증 API입니다.

**흐름:** 인증 코드 발송 → 인증 코드 확인 → 회원가입

---

## 1. POST `/api/email/verification/send` — 인증 코드 발송

입력한 이메일로 6자리 인증 코드를 발송합니다.

### Request

#### Headers

| Key | Value |
|-----|-------|
| `Content-Type` | `application/json` |

#### Body

```json
{
  "email": "user@example.com"
}
```

#### 필드 설명

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `email` | `String` | Y | 인증할 이메일 (이메일 형식) |

### Response

#### 성공 (HTTP 200)

```json
{
  "success": true,
  "code": "200",
  "message": "인증 코드가 발송되었습니다.",
  "data": null
}
```

#### 에러 코드

| HTTP Status | message | 발생 상황 |
|-------------|---------|-----------|
| `400` | 잘못된 입력값입니다. | 이메일 누락 또는 이메일 형식 오류 |
| `500` | 이메일 발송에 실패했습니다. | SMTP 등 외부 이메일 발송 오류 |
| `500` | 서버 내부 오류가 발생했습니다. | 서버 오류 |

---

## 2. POST `/api/email/verification/verify` — 인증 코드 확인

발송된 인증 코드를 입력해 이메일 소유를 검증합니다.

### Request

#### Headers

| Key | Value |
|-----|-------|
| `Content-Type` | `application/json` |

#### Body

```json
{
  "email": "user@example.com",
  "code": "482719"
}
```

#### 필드 설명

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `email` | `String` | Y | 인증 코드를 발송한 이메일 |
| `code` | `String` | Y | 수신한 6자리 인증 코드 |

### Response

#### 성공 (HTTP 200)

```json
{
  "success": true,
  "code": "200",
  "message": "이메일 인증이 완료되었습니다.",
  "data": null
}
```

#### 에러 코드

| HTTP Status | message | 발생 상황 |
|-------------|---------|-----------|
| `400` | 잘못된 입력값입니다. | 이메일/코드 누락, 코드가 6자리가 아닌 경우 |
| `400` | 인증 코드가 만료되었습니다. | 인증 코드 유효 시간 초과 |
| `400` | 인증 코드가 올바르지 않습니다. | 코드 불일치 |
| `404` | 인증 요청을 찾을 수 없습니다. | 해당 이메일로 발송된 인증 요청이 없는 경우 |
| `500` | 서버 내부 오류가 발생했습니다. | 서버 오류 |

---

## 주의사항

- 인증 코드는 **6자리 숫자**입니다.
- 인증 코드에는 유효 시간이 있습니다. 만료 시 `/send`를 다시 호출하여 새 코드를 발급받아야 합니다.
- 인증 완료(`verify` 성공) 후에도 일정 시간이 지나면 인증이 만료됩니다. 만료 전에 회원가입을 완료해야 합니다.
- 동일 이메일로 `/send`를 여러 번 호출하면 가장 마지막에 발송된 코드만 유효합니다.
